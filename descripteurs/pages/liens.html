⚠️ Les fenêtres pop-up sont bloquées

Kizako a besoin d'ouvrir deux fenêtres côte à côte pour fonctionner. Voici comment autoriser les pop-ups pour ce site :

1 Regarde la barre d'adresse de Chrome, tout à droite. Tu verras un petit carré barré.
2 Clique sur ce petit carré, puis choisis "Toujours autoriser les pop-ups de kizako.ch".
3 La page se recharge. Clique de nouveau sur le lien.

Si le petit carré n'apparaît pas ou que l'option est grisée, Chrome est administré par l'école et bloque les pop-ups. Essaie d'utiliser Firefox.

1 Regarde la barre d'adresse de Edge, tout à droite. Tu verras un petit carré barré.
2 Clique sur ce petit carré, puis choisis "Toujours autoriser les pop-ups de kizako.ch".
3 La page se recharge. Clique de nouveau sur le lien.

Si le petit carré n'apparaît pas ou que l'option est grisée, Edge est administré par l'école et bloque les pop-ups. Essaie d'utiliser Firefox.

1 Une barre de notification apparaît en haut du navigateur : "Firefox a bloqué l'ouverture d'une fenêtre pop-up".
2 Clique sur "Préférences" dans cette barre, puis sur "Autoriser les pop-ups de kizako.ch".
3 Firefox ouvre automatiquement les fenêtres. Si ce n'est pas le cas, clique de nouveau sur le lien.
1 Ouvre les paramètres de ton navigateur et cherche la section "Pop-ups" ou "Fenêtres".
2 Autorise les pop-ups pour kizako.ch, puis recharge la page et réessaie.
